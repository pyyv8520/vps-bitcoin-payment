## 为什么要用 Bitcoin 购买 VPS？

说实话，大多数人选择用比特币付款不是因为技术极客情结，而是有几个很现实的理由：

**1. 隐私保护**

信用卡付款会把你的姓名、账单地址、消费记录留在银行流水里。比特币交易虽然链上可查，但不需要把个人身份信息和交易绑定在一起。配合一个隐私邮箱注册账号，整个购买流程能做到相当干净。

**2. 跨境支付无障碍**

有些地区的信用卡在国际支付时会遇到风控拦截，或者需要额外的 3DS 验证。比特币完全绕开这套系统，转账就是转账，没有中间层来 reject 你的支付。

**3. 去中心化，不依赖银行**

部分用户的业务场景涉及特定行业，银行账户存在被限制的风险。加密货币支付不经过传统金融体系，稳定性更可预期。

**4. 透明结算，没有隐藏费用**

这一点要结合具体服务商，但像 Evoxt 这样标榜"transparent pricing"的平台，用比特币付款和用信用卡付款是完全一样的价格，没有额外手续费。
